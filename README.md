# Автоматизированное рабочее место (АРМ) «Складской учет»

Программный комплекс предназначен для автоматизации процессов учета материально-технических ценностей на складах предприятий. Разработан в рамках курсового проекта по дисциплине «Технологии программирования».

## Основной функционал системы
* **Учет складских запасов:** Регистрация прихода новых товаров, аналитика текущих остатков в реальном времени.
* **Документооборот:** Формирование расходных и приходных накладных с автоматической генерацией уникальных номеров документов.
* **Бизнес-валидация:** Защита от проведения некорректных операций (блокировка отгрузки при нехватке товара на балансе).
* **Экспорт отчетов:** Формирование инвентарных ведомостей и выгрузка структурированных данных в текстовые файлы.
